

settings.txt - файл с параметрами
main.py - программа для расчета траектории 3-ех тел и их визуализация
analysis.py - программа для просмотра уже просчитанных траекторий

В settings.txt содержаться все параметры которые можно изменить.

При изменении параметров делать по аналогии с уже имеющимися.

Число итераций и временной шаг имеют целочисленный тип

Записывать данные можно сделающим образом: 60_000, 60000, 6e4

При выборе числа итераций:

Число итераций   |   Время выполения
--------------------------------------
   10_000	     |  порядка 30 секунд
   20_000        |  порядка 1 минут
   30_000	     |  порядка 2 минут
   60_000        |  порядка 10 минут
  100_000        |  порядка 2/3 - 1 часа
  200_000        |  порядка 4 часов


Данные записываются в папке "день-месяц-год час-минута"
При запуске программы в то время которое уже имеется, файлы будут перезаписаны

При возникновении ошибки в analysis.py: Redirection is not supported.
Следует эмулировать терминал во внешней консоли. Если используете PyCharm зайдите Run Configurations, затем Modify options и поставьте галочку Emulate terminal in output console.
